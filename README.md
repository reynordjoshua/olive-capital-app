# Olive Capital — Autonomous Financial Analyst

A single-page tool that computes financial ratios, runs a DCF valuation, and
generates a written analyst note — entirely client-side, no backend or API
key required.

## Project contents

```
olive-capital-app/
├── index.html      ← the entire app (HTML + CSS + JS, self-contained)
├── vercel.json      ← deployment config
└── README.md         ← this file
```

## 1. Install on your laptop

1. Download this folder (or the zip) and unzip it.
2. Move the unzipped `olive-capital-app` folder to `D:\olive-capital-app`
   (or wherever you keep projects).

## 2. Open in VS Code

1. Open VS Code.
2. `File → Open Folder…` → select `D:\olive-capital-app`.
3. To preview it locally before deploying:
   - Install the **Live Server** extension (search it in the Extensions
     panel, install).
   - Right-click `index.html` in the file explorer → **Open with Live
     Server**. It opens in your browser at `http://127.0.0.1:5500`.
   - Or simply double-click `index.html` in File Explorer to open it
     directly in a browser — it works without a server too.

## 3. Deploy to Vercel

You have two options — pick whichever you're more comfortable with.

### Option A — Vercel CLI (fastest, done from VS Code's terminal)

1. In VS Code, open a terminal: `Terminal → New Terminal`.
2. Install the Vercel CLI (one-time):
   ```
   npm install -g vercel
   ```
3. Log in (opens a browser to authenticate):
   ```
   vercel login
   ```
4. Deploy from inside the project folder:
   ```
   vercel
   ```
   Answer the prompts:
   - "Set up and deploy?" → `Y`
   - "Which scope?" → your account
   - "Link to existing project?" → `N`
   - "What's your project's name?" → `olive-capital` (or anything you like)
   - "In which directory is your code located?" → `./`
   - It auto-detects a static site — accept the defaults.
5. It gives you a live URL immediately (e.g.
   `https://olive-capital.vercel.app`). That's your shareable link for
   your resume/LinkedIn.
6. Every time you make changes, redeploy with:
   ```
   vercel --prod
   ```

### Option B — Vercel dashboard (no CLI, drag-and-drop)

1. Go to https://vercel.com and sign in (GitHub, GitLab, or email).
2. Click **Add New… → Project**.
3. Choose **"Deploy without Git"** / drag-and-drop, and drop the whole
   `olive-capital-app` folder in.
4. Click **Deploy**. You'll get a live URL in ~30 seconds.

## 4. (Optional) Put it on GitHub first

If you want version history and easier redeploys:

```
cd D:\olive-capital-app
git init
git add .
git commit -m "Olive Capital financial analyst tool"
```
Then create a new repo on GitHub, follow its "push an existing repo"
instructions, and connect that repo to Vercel from the dashboard
(**Add New… → Project → Import Git Repository**) — after that, every
`git push` auto-deploys.

## Notes

- The app is 100% client-side — all ratio math, DCF calculations, and the
  analyst commentary are generated by JavaScript in the browser. Nothing
  is sent to a server, so it works the same locally and once deployed.
- The logo is embedded directly in `index.html` as base64, so the whole
  site is one portable file — you can rename or move it freely.
